# EXECUTAR


## Ative a virtual env:

```bash

source ./bin/venv/activate #linux

```

## Para executar, rode primeiramente o servidor

- Com o virtual env ativado. Execute o servidor:

```bash

python3 ./server.py

```

- Abra outros dois terminais, execute pelo menos dois clientes:

```bash

python3 ./client.py

```

```bash

python3 ./client.py

```
